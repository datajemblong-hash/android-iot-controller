# Name: Interactive Error Solver
# Description: Skill untuk menganalisis log error, menemukan letak kesalahan di kode proyek, dan merumuskan langkah perbaikan secara instan.

## What I do
Saya membantu pengguna mengatasi error teknis (runtime error, crash, compile error, dll.) yang terjadi selama pengerjaan proyek. Saya membaca kode yang relevan dengan pesan error, mencari akar masalahnya, dan langsung merumuskan perbaikan kode.

## When to use me
- Ketika program/aplikasi tiba-tiba crash atau berhenti bekerja.
- Ketika terminal menampilkan pesan error berwarna merah yang tidak Anda pahami.
- Ketika hasil keluaran program tidak sesuai dengan yang tertulis di PRD.

## How I do it
1. Minta pengguna mengirimkan:
   - Salinan pesan error lengkap dari terminal/browser console/log device.
   - Penjelasan singkat tentang apa yang sedang dicoba ketika error muncul.
2. Cari nama file dan nomor baris yang disebutkan di dalam pesan error (stack trace).
3. Buka dan baca file kode yang bermasalah tersebut.
4. Analisis penyebab error tersebut (misal: variabel belum diinisialisasi, salah tipe data, koneksi database gagal, dll.).
5. Berikan penjelasan sederhana mengenai penyebab error (tanpa bahasa pemrograman yang terlalu rumit).
6. Tuliskan perbaikan kode yang tepat secara utuh, bukan potongan kode agar pengguna tinggal menyalinnya.
7. Setelah perbaikan diterapkan, pandu pengguna untuk menjalankan ulang aplikasi dan memverifikasi apakah error sudah hilang.
8. Laporkan: "🛠️ Error berhasil dianalisis dan diperbaiki!"
